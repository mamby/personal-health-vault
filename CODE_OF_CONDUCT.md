# Code of Conduct

## Our Commitment

We aim to keep this project open, respectful, and welcoming to everyone.

## Expected behavior

- Be respectful and considerate
- Focus on constructive feedback
- Assume good intent in discussions
- Keep communication professional and relevant
- Respect differing opinions and experience levels

## Unacceptable behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Trolling, spam, or disruptive conduct
- Sharing private information without permission

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any related communication.

## Enforcement

Maintainers may:

- Request changes in behavior
- Remove or edit content
- Temporarily or permanently restrict access to the project

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers through a private channel (e.g. email or direct message if available).

All reports will be handled confidentially and fairly.

---

Thank you for helping keep this project respectful and productive.
